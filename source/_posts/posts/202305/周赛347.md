### 2710. 移除字符串中的尾随零

![image-20230530090649271](https://cdn.gooohlan.cn/img/202305300906292.png)

#### 示例代码：

```go
func removeTrailingZeros(num string) string {
    b := []byte(num)
    i := len(b)
    for ; i > 0 && num[i] == '0'; i-- {
    }
    return string(b[:i+1])
    
    // return strings.TrimRight(num, "0") // 库函数写法
}
```
