题目: 2663. 字典序最小的美丽字符串

题解和思路

「不包含任何长度为 222 或更长的回文串」，如果一个字符串是回文串，那么去掉首尾它依旧是回文串，直到最后回文串的最终长度为 222 或者 333，此时等价于「不包含长度为 222 或者 333 」d 的回文串。也即是说，不能出现 s[i]==s[i−1]s[i] == s[i-1]s[i]==s[i−1] 以及 s[i]==s[i−2]s[i] == s[i-2]s[i]==s[i−2]。

根据此性质，只需要判断 s[i]s[i]s[i] 左侧的两个字母即可。s[i]s[i]s[i] 极其右侧的字母交给它右侧的字母来判断

要求字典序最小，修改最右侧就可以满足。

例如： s=bcaz,k=26s=bcaz,k=26s=bcaz,k=26，答案需要比 sss 大，先将 s[3]=zs[3]=zs[3]=z 加一，因为 z+1>kz+1>kz+1>k，所以需要进位得到 bcbabcbabcba。但是这样前三个字母 bcbbcbbcb 形成了回文串，接着处理 bcbbcbbcb，把 s[2]=bs[2]=bs[2]=b 加一，得到 bccabccabcca，此时中间两个字符串形成了回文串 cccccc，将 s[2]=cs[2]=cs[2]=c 加一，得到 bcdabcdabcda，此时已经没有回文串了。

题目保证了输入的 sss 是美丽字符串（不包含回文串），所以一旦发现此字符左侧不存在回文串时我们往右寻找，也不存在就可以返回答案。当 s[0]s[0]s[0] 加一后不在前 kkk 个字母中的情况，说明答案为空，返回空字符串。

题目: 520. 检测大写字母

题解和思路

遍历所有字符，统计大写字母数量 countcountcount，以下三种情况均为 truetruetrue：

  • count=0count=0count=0，全是小写字母
    • count=len(word)count = len(word)count=len(word)，全部为大写字母
    • count=1count=1count=1 且 word[0]word[0]word[0] 即首字母是大写

unicode.IsUpper(s) 可以判断 s 是否为大写字母

func detectCapitalUse(word string) bool {
    count := 0
    for _, s := range word {
        if unicode.IsUpper(s) {
            count++
        }
    }
    
    return count == 0 || count == len(word) || count == 1 && unicode.IsUpper(rune(word[0]))
}
